# WhatsApp Auto Send Helper

A lightweight Tampermonkey script for WhatsApp Web.

This script automatically:

* Detects the WhatsApp Send button
* Clicks the Send button automatically
* Handles invalid number popups
* Closes the browser tab after completion

## What This Repository Includes

✅ Tampermonkey Helper Script

## What This Repository Does NOT Include

❌ Bulk Messaging Logic

❌ Contact List Management

❌ Campaign Management

❌ Marketing Automation Workflow

The script available in this repository is only a helper component used within a larger WhatsApp automation workflow.

## Installation

1. Install Tampermonkey Extension
2. Create a New Script
3. Copy the code from `whatsapp-auto-send-helper.user.js`
4. Save the script
5. Open WhatsApp Web
